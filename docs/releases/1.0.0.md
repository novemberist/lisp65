# lisp65 1.0.0 Release Notes

Released: 2026-07-15
Language profile: Dialect V2
Proof source tag: `v1.0.0`
Proof source commit: `589729471b39ef218397663480905a36fd03dd16`

The curated public repository has independent snapshot history. Its public
`v1.0.0` tag identifies the reviewed source distribution, while the manifest
and release receipt retain the authoritative private proof-commit identity.

## What shipped

- Native MEGA65 Lisp REPL
- On-device `lcc` bytecode compiler
- Closures, macros, higher-order functions, and strict arity
- Full-screen Emacs-style editor
- On-demand IDE and optional IDEX feature tiers
- Source evaluation and persistent L65M library compilation
- Transactional M65D create/replace persistence
- Autonomous cold boot from `L65SYS`
- A blank `L65WORK` convenience image; any valid non-product 1581 disk is valid

Product version 1.0.0 intentionally contains Dialect V2. Dialect V1 was an
internal migration baseline and was never a product release.

- G5 global hardware matrix: 14/14 passed
- G6 hardware boot/media profile: 5/5 applicable cases passed
- Physical product-medium write protection: not applicable to the stock-core
  SD-D81 profile used for G6

## Known limitations

- One-drive, two-media workflow
- No on-device disk formatter
- No undo/redo in the editor
- No full ANSI Common Lisp compatibility
- No atomicity claim for the measured Freezer race after a sector write has
  already started; at most one sector may cross the media boundary before the
  terminal guard stops subsequent writes

## Verification

The release bundle is self-contained and verified with:

```sh
python3 verify.py
```

Its manifest includes the product artifact SHAs, toolchain provenance, gate
claims, source seal, packer, and offline verifier. The release build was
reproduced under varied environment settings and all 13 product artifacts are
byte-identical to the G6-sealed product.

The public source repository carries a compact
[artifact manifest](../../releases/lisp65-1.0.0-manifest.json) with the exact
SHA-256 digest and byte count of each of those 13 artifacts. The full bundle is
distributed as a GitHub Release asset rather than through Git LFS.
